# Учёт книг в библиотеке (C++)

## Описание проекта
Данный проект представляет собой лабораторную работу по информатике, посвящённую созданию простой системы учёта библиотечных книг. Система позволяет добавлять книги, хранить информацию об авторах, отслеживать их доступность, а также фиксировать выдачу и возврат читателям.

## Цель проекта
Автоматизировать процесс ведения библиотеки, чтобы ускорить поиск книг, управление фондами и анализ состояний книг.

## Функциональные возможности системы
- Добавление, изменение и удаление книг
- Учёт авторов
- Регистрация выдачи и возврата книг
- Просмотр доступных и выданных книг

## Используемый язык программирования
C++

## Постановка задачи
Каждая книга имеет автора и статус (доступна или выдана). Читатель может брать несколько книг, каждая выдача имеет дату выдачи и дату возврата. Выдачу и возврат выполняет только библиотекарь. Читатель может только просматривать список доступных книг.

## Как запускать проект
1. Перейти в папку src
2. Скомпилировать файл LibraryApp.cpp с помощью g++
3. Запустить полученный исполняемый файл

Пример:
bash
g++ LibraryApp.cpp -o library
./library
